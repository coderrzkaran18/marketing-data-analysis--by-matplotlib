# Marketing Strategy & Sales Analysis (Using Python & Matplotlib)

This project analyzes a real-world retail sales dataset to generate business insights using Python, Pandas, and Matplotlib.  
The goal is to identify profitable regions, customer segments, and product categories through data visualization and trend analysis.

## Key Analysis
- Region-wise sales and profit analysis  
- Customer segment performance  
- Product category profitability  
- Sales vs profit relationship  
- Monthly sales trend (time series)  
- Final dashboard using Matplotlib  

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook (VS Code)

## Outcome
Identified high-performing regions, valuable customer segments, and profitable product categories to support marketing decision-making.  
Created a visual dashboard to present insights clearly.
